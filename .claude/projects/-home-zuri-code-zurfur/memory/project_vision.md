Zurfur is a decentralized art commission platform built on AT Protocol (Bluesky). Core vision: a "Super App" for the creator economy — consolidates commissions, invoicing, chat, and profiles with data sovereignty via AT Protocol.

**Why:** solves trust, data-ownership, and workflow-fragmentation problems in the art-commission space; AT Protocol integration ensures users own their data and reputation.

**Single source of truth = Confluence DESIGN, not in-repo docs.** There is no `design/` directory. Scope/roadmap live in **Project MVP** (Confluence page `589826`, intentionally in flux).
